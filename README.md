## Installation and Setup

### 1. Clone this repo
`git clone https://github.com/harshawasthi512/`

### 2. Navigate to the backend directory and run backend
 - `cd genai/backend`
 - `pip install -r requirements.txt`
 - `uvicorn main:app --reload` 

### 3. Backend will run on the port 8000

### 4. Open you browser and click on three dots and click on extension tab -> Manage Extension.
### 5. Switch on the developer mode.
### 6. Click on Load unpacked and select the frontend directory from the cloned repo
### 7. Now, it will work seamlessly with Gmail.
